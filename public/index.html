Image Search Abstraction Layer

  • User Story: I can get the image URLs, alt text and page urls for a set of images relating to a given search string
  • User Story: I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • User Story: I can get a list of the most recently submitted search strings.

Usage:

To search for images, send a request to "https://fcc-imagsearch.herokuapp.com/search?q=[searchString]&offset=[offset]", where [searchString] is the search term and offset is an optional parameter to paginate results

To list the most recently submitted search strings, send a request to https://fcc-imagsearch.herokuapp.com/search
